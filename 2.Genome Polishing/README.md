## 2. Genome Polishing
This repository contains a configuration and workflow for genome polishing using [NextPolish (version 1.4.1)](https://github.com/Nextomics/NextDenovo), integrating both short-read (SGS) and long-read (LGS) sequencing data to improve assembly accuracy.

Input your files and parameters, then run the software with this command

    `nextPolish nextpolish_config.txt`
