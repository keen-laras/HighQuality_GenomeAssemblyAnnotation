## 1. Genome Assembly
This repository contains a configuration and workflow for de novo genome assembly using [NextDeNovo (version 2.5.2)](https://github.com/Nextomics/NextDenovo), optimized for long-read sequencing data such as CycloneSEQ and Oxford Nanopore.

Input your files and parameters, then run the software with this command

    NextDenovo 1.nextdenovo_config.txt
