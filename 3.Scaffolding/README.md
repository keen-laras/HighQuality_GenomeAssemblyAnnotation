## 3. Genome Scaffolding 
This repository contains a pipeline for genome scaffolding using chromatin conformation capture data (Hi-C / Pore-C), powered by [YaHS](https://github.com/c-zhou/yahs), [Juicer](https://github.com/aidenlab/juicer) & [Juicer_tools](https://github.com/aidenlab/JuicerTools) for visualization and manual curation.

Run alignments using,

`bash run_bowtie2.sh` then `run scaffolding_script1.sh`

or

`bash run_chromap.sh` then `run scaffolding_script2.sh`
